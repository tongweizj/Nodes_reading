# React + Vite

demo 用于展示react 如果使用web浏览器的存储机制

## 运行

```
npm i
npm run dev
```

http://localhost:5173/

## 实现功能

1. 输入同时，存储在本地
2. 刷新页面，数据不丢
3. 自定义hook，封装了功能
